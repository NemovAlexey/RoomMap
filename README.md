RoomMap - web-приложение для отображения карт помещений или любой другой графики с возможностью подсветки и описания областей карты. 
Вы можете создавать описания помещений/офисов/комнат или, например, узлы какого-либо механизма и т.п.

Приложение позволяет:
1) Отображать карты неограниченного размера
2) Загружать карту частями (только видимую область)
3) Создавать многоуровневые карты (карты многоэтажных строений)
4) Накладывать информационные слои на карту
5) Создавать информационные области на карте (полигоны и окружности), для описания отдельных объектов.

Приложение корректно работает во всех современных браузерах, в том числе и мобильных.

Распространяется по лицензии GPL.
